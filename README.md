### run application
```
cd hello
go run hello.go or go run .
```

### build application
```
go build hello.go
```
### after build
```
./hello
```



## simple git command
### create new branch
```
git branch <new-branch-name>
```
### create new branch and switch
```
git checkout -b <new-branch-name>
```
### merge
```
git merge <branch-name>
```
### delete branch
```
git branch -D <branch-name>
```
